# page-inf

### DESCRIPTION :
Page-inf is a tool for web analysis, basic but in my opinion it can be useful.

#### FUNCTIONS
<ul type="square">
  <li>scan ip addresses</li>
  <li>scan ports</li>
  <li>scan possible entries to the admin panel</li>
</ul>
